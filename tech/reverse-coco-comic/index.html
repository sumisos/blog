前言

遇事不决看刑法

2021 年 3 月 1 日开始生效的《中华人民共和国刑法修正案（十一）》：

二十、将刑法第二百一十七条修改为：“以营利为目的，有下列侵犯著作权或者与著作权有关的权利的情形之一，违法所得数额较大或者有其他严重情节的，处三年以下有期徒刑，并处或者单处罚金；违法所得数额巨大或者有其他特别严重情节的，处三年以上十年以下有期徒刑，并处罚金：
“（一）未经著作权人许可，复制发行、通过信息网络向公众传播其文字作品、音乐、美术、视听作品、计算机软件及法律、行政法规规定的其他作品的；
“（二）出版他人享有专有出版权的图书的；
“（三）未经录音录像制作者许可，复制发行、通过信息网络向公众传播其制作的录音录像的；
“（四）未经表演者许可，复制发行录有其表演的录音录像制品，或者通过信息网络向公众传播其表演的；
“（五）制作、出售假冒他人署名的美术作品的；
“（六）未经著作权人或者与著作权有关的权利人许可，故意避开或者破坏权利人为其作品、录音录像制品等采取的保护著作权或者与著作权有关的权利的技术措施的。”

二十一、将刑法第二百一十八条修改为：“以营利为目的，销售明知是本法第二百一十七条规定的侵权复制品，违法所得数额巨大或者有其他严重情节的，处五年以下有期徒刑，并处或者单处罚金。”

然后开始

起初是我姐突然让我帮忙找下《天官赐福》的漫画，看了下现在的版权在 阿 B 手里。
我寻思应该是个大会员的事吧，一看跟大会员没有一毛钱关系，阿 B 的漫画站是独立运营的，行 8。
免费到 18 话，《天官赐福》只支持整卷购买，而且想付费还要必须下手机 APP，您配钥匙吗。

首先我不是美化自己的行为，我以个人娱乐为目的使用盗版虽然没有犯罪，但根据普世价值观显然是不道德的行为。
我就是盗版狗，该骂就骂。

接着我再说我的观点，动漫之家被高限处罚还是上个月的事（这里腾讯新闻用看热闹不嫌事大的语气相当戏谑地陈述了相关事件）。
我这里就不阴谋论「肯定是阿 B 举报的」了，B 漫的吃相继承了阿 B 一贯的「优良」历史传统，就凭你这个吃相，我说「肯定不关阿 B 的事我们小破站是无辜的白莲花」不知道有没有人会信。

字幕组

字幕组是一个灰色地带，但不管怎么灰色，「购入图源 / 扫图 / 翻译 / 修图 / 嵌字」整套流程走下来也不是什么轻松的活，有时候确实有金主赞助，但很多时候字幕组都是用爱发电的，硬要算的话人家怎么说也是公益组织。
白嫖归白嫖，但实际跟钱其实关系不大，大家更在乎的都是内容，你画得不好看谁 TM 闲得蛋疼给你汉化；画一堆厕纸倒贴钱请人看，人家都未必愿意抽出这个时间。
最典型的例子：《异世界叔叔》在动漫之家反响很好，于是字幕组去联系的作者，跟他说中国有很多读者喜欢你的漫画，但我们就是一群屁民没那个能力帮你正规引进，能不能弄个途径方便我们打钱。
所以在我看来，字幕组是对原作者是寄生关系，但大部分时候是迫于无奈（「一刀切 = 懒政」是中央定性的，不是我乱说的。但我们就是不做分级制度ᕕ( ᐛ )ᕗ有可能影响到未成年就是一刀切），而且有时候是能够反哺原作者的。因为对我们这代人来说，给当年的情怀「补票」想必是一件很普遍的雅事了。

阿 B 的吃相

B 漫先是砸钱拿下人气高的漫画版权（还不用我们自己出市场调查的成本，反正你们动漫之家有现成的排行榜。商业道德？什么商业道德，能吃吗），然后堂而皇之地把字幕组的用爱发电的成品偷过去了，反正你们都已经在夹缝里求生存了，你小小字幕组敢跟我们对着干吗。(　ˇωˇ)鲁迅知道你们是这样用「拿来主义」的吗？
被声讨之后，B 漫做了一个更离谱的操作，它把原汉化组精心翻译校对的字幕涂了，然后用机翻重新汉化了一遍，机翻的质量网上很多评价我就不说了。这套操作背后的商业逻辑是「反正我们全网独家，爱看看不看滚」。根据这套逻辑，所以 B 漫的定价也是相当放飞自我的，颇有破罐破摔的意思：我摊牌了，我不装了，我就是吃相难看，你能怎么着吧。
B 漫具体是怎么起家的，即使陈叔叔公关这一块拿捏得再死，互联网也是有记忆的，更多详情有兴趣可以自己搜。

盗版站

盗版怎么洗都洗不白，看盗版是不道德的，盗版站更不是什么好东西。
他们以提供免费资源为饵，吸引因为各种原因不想付费的用户来观看，靠（擦边球或者直球）违法的广告攫取私益。
这种行为更是明确触犯了上面提到的第二百一十七条刑法。

所以这件事三方都不是什么好东西，版权方（你 B）、盗版站、使用盗版的我。

function decryptAES(data, key) {
    let iv = "";
    let clearEncoding = 'utf8';
    let cipherEncoding = 'base64';
    let cipherChunks = [];
    let decipher = crypto.createDecipheriv('aes-128-ecb', key, iv);
    decipher.setAutoPadding(true);

    cipherChunks.push(decipher.update(data, cipherEncoding, clearEncoding));
    cipherChunks.push(decipher.final(clearEncoding));

    return cipherChunks.join('');
}

function decodeAndDecrypt(value, keyArr) {
    let decodedValue = Buffer.from(value,'base64').toString();

}
