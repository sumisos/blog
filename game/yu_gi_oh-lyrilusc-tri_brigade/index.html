LL 铁比起纯铁，就是多了 LL 这条以展开为核心的轴。
相较之下 LL 铁偏向于展爆思路；纯铁偏向于运营思路。

增加了大量特召点（来自 LL），导致即使在吃坑被打断以后也能做出可观的妥协场，不至于直接歇逼。
这大幅提高了卡组的容错率。

然而缺点也被放大了：吃怪兽康、终端缺少魔陷阻抗、无法解决后场。
尤其是技抽，见到直接 run。

至于干扰能力，大部分是封锁和弹手型阻抗。

硬要说强度的话，LL 铁是不如十二铁和纯铁的。为什么？
十二铁启动（即使是后攻）只需要通召（不入任何连锁）一个怪（马剑），通召完一口气（中间不需要发任何效果）叠出一个四素材或者六素材高达，然后就能用天霆号把你的场和康全部逼停，后攻突破很强。
LL 铁后攻是很难突破，而且先攻吃到陨石是很难受的，陨石 / 灰 / G 都很难顶。

对局技巧 Tips

  • 人马 可以堆墓任何 LL 小鸟，然后特召 小绿 或者 小黄 拉回场。
  • 同时带了两种鸟的构筑，王神鸟 优先拉 巨神鸟（全康）还是 结界像（封锁特召）？
    做 小琉璃 的场面是没有抗性的，抛开 小棕 不谈，全家的抗性都在 聚集夜莺 那。
    没有手坑的话拉 结界像 是非常冒险的行为（吃一个 电爆 就炸了，辛苦半天做好（说书）的场上只剩有双抗的 龙皇）。
    因此手里有「反制手坑 / 大量启动点」才考虑拉 结界像 一劳永逸地锁死特召（永续效果，双方都不能特召风属性以外的怪）；心里没底还是稳妥点拉 巨神鸟 吧。
  • 两套主轴如何联动（互相拉起）？
    只有「鸟」的情况：请记住 纳贝尔 是 1 星鸟兽族（能被 小棕 从卡组检索，也能被 小蓝 从手牌特召），亡语（进墓触发）检索「铁兽」字段。
    只有「兽」的情况：就堆墓，然后靠「铁兽」的本家通用天赋（除外墓地作为代价）LINK 出 凶鸟（LINK-4，亡语检索）或者 徒花（LINK-2，M1 发效果从手牌特召）。

常用定式 Combo

以下 combo 需要注意一点：绿柱石金丝雀 的自肃。
如果发动了拉墓地的效果，从额外卡组只能召唤 超量怪兽（XYZ） 而无法召唤 连接怪兽（LINK）。
所以小黄（绿柱石金丝雀，重点是金不是绿）一般最后才拉，作为定场的这么一个存在。

人马展开

人马 + 任一「兽族 / 兽战士族 / 鸟兽族」= 王神鸟 + 龙皇 + 四素材小琉璃 + 巨神鸟/结界像

  1. 弗拉克杜尔(弩人马) 发效果堆墓 姬特(兔子?)
  2. 姬特 进墓发效果堆墓 纳贝尔(投弹鸟) 拿 克拉斯(RPG 大角牛)
  3. 克拉斯 丢任一「兽族 / 兽战士族 / 鸟兽族」特招上场
  4. 克拉斯 发效果除外墓地 4 只怪兽出 凶鸟之施莱格
  5. 把场上的 克拉斯 和 凶鸟之施莱格 LINK 出 徒花之费莉吉特
  6. 凶鸟之施莱格 进墓发效果检索 钴尖晶雀(棕鸟)
  7. 徒花之费莉吉特 发效果下 钴尖晶雀
  8. 钴尖晶雀 检索 天青石鹡鸰(灰鸟)
  9. 通常召唤 天青石鹡鸰
  10. 钴尖晶雀 和 天青石鹡鸰 叠一只 吟诵椋鸟
  11. 吟诵椋鸟 发效果丢 天青石鹡鸰 检索 绿柱石金丝雀(黄鸟)
  12. 徒花之费莉吉特 和 吟诵椋鸟 LINK 出 王神鸟 斯摩夫
  13. 绿柱石金丝雀 发效果拉墓地 天青石鹡鸰
  14. 天青石鹡鸰 发效果检索 鸟鸣呼唤
  15. 绿柱石金丝雀 和 天青石鹡鸰 叠第二只 吟诵椋鸟
  16. 吟诵椋鸟 丢 天青石鹡鸰 检索 绿松石莺(绿鸟)
  17. 鸟鸣呼唤 检索 青玉燕(蓝鸟) 特招 绿松石莺
  18. 绿松石莺 发效果拉墓地里的第一只 吟诵椋鸟
  19. 墓地的 天青石鹡鸰 发效果给拉回来的 吟诵椋鸟 塞素材
  20. 拉回来的 吟诵椋鸟 再发效果检索 纳贝尔
  21. 两只 吟诵椋鸟 做 未来 No.0 未来皇 霍普 然后叠 未来 No.0 未来龙皇 霍普
  22. 青玉燕 发效果带 纳贝尔 跳下来
  23. 将场上的 绿柱石金丝雀、青玉燕、纳贝尔 叠成 合唱小琉璃
  24. 青玉燕 发效果给 合唱小琉璃 叠一个素材
  25. 回合结束，王神鸟 斯摩夫 发效果拉 霞之谷的巨神鸟

绿棕展开

小绿 + 小棕 = 龙皇 + 弗拉克杜尔 + 王神鸟 + 四素材小琉璃 + 巨神鸟/结界像

  1. 绿松石莺 发效果带 钴尖晶雀 跳下来
  2. 钴尖晶雀 发效果检索 纳贝尔
  3. 绿松石莺 和 钴尖晶雀 叠一只 吟诵椋鸟
  4. 吟诵椋鸟 丢 绿松石莺 检索 青玉燕
  5. 青玉燕 发效果带 纳贝尔 跳下来
  6. 青玉燕 和 纳贝尔 叠第二只 吟诵椋鸟
  7. 第二只 吟诵椋鸟 丢 纳贝尔 检索 天青石鹡鸰
  8. 纳贝尔 进墓发效果检索 弗拉克杜尔（别的也行，但人马身材最好，1900 / 1600）
  9. 两只 吟诵椋鸟 做 未来 No.0 未来皇 霍普 然后叠 未来 No.0 未来龙皇 霍普
  10. 通常召唤 弗拉克杜尔 发 ② 效果（① 是堆墓）除外墓地 2 只怪兽出 徒花之费莉吉特
  11. 徒花之费莉吉特 发效果下 天青石鹡鸰
  12. 天青石鹡鸰 发效果检索 鸟鸣呼唤
  13. 徒花之费莉吉特 和 天青石鹡鸰 LINK 出 王神鸟 斯摩夫
  14. 鸟鸣呼唤 检索 绿柱石金丝雀 随便下什么（绿 / 棕 / 蓝 / 灰 / 纳贝尔都用过了）
  15. 绿柱石金丝雀 发效果拉墓地 青玉燕
  16. 将场上的 绿柱石金丝雀 和 青玉燕 叠成 合唱小琉璃
  17. 青玉燕 发效果给 合唱小琉璃 叠一个素材
  18. 墓地的 天青石鹡鸰 发效果叠给 合唱小琉璃（第 14 步 鸟鸣 检索完如果下了鸟是五素材）
  19. 回合结束，王神鸟 斯摩夫 发效果拉 霞之谷的巨神鸟
