本家主轴 半龙女仆

龙女仆本家怪兽的通用特性：

  • 女仆装的小体型（人形态）（下级怪兽）有两个类似的效果
    • ①：通召 / 特召成功（战吼）有一个效果，根据不同职务（颜色）区别各异
      • 育婴女仆
        （小粉）：苏生一只别的女仆
      • 客厅女仆
        （小绿）：从卡组堆墓一张别的女仆
      • 蒸馏室女仆
        （小红）：从卡组检索一张别的女仆，然后丢一张本家怪兽
      • 寝室女仆
        （小黑）：从卡组检索一张本家魔陷
    • ②：双方 进战阶可以弹回手卡（变身本体），然后从手卡 / 墓地特召一只本家大怪（龙形态）（上级怪兽）
  • 本体的大体型（龙形态）（上级怪兽）有三个类似的效果
    • ①：从手卡丢弃有一个效果，根据不同职务（颜色）区别各异
      • 地慈龙女
        （大粉）：从手卡特召一只女仆（手坑）
      • 天风龙女
        （大绿）：指定一只怪兽不能发效果
      • 赤焰龙女
        （大红）：指定一只本家怪兽 + 2000 ATK（手坑）
      • 没有大黑
    • ②：只要自己场上存在融合怪兽，获得效破抗性（不会被效果破坏）
    • ③：双方 战阶结束可以弹回手卡（变身女仆），然后从手卡特召特召一只本家小怪（人形态）（下级怪兽）

从手卡 / 墓地特召一只星级 ± 1 星的本家怪兽守备表示
②：己方其他龙族怪兽回手时可以破坏对方一只怪兽（取对象）
耀光龙女

（大姐，10 星）		①：双方 的准备阶段，从手卡 / 墓地特召一只（9 星以下）本家怪兽
即耀光以外的任何本家怪，包括二姐
②：对方发动魔法 / 陷阱 / 怪兽效果时，无效并破坏，变回女仆形态
即耀光（大姐）返回额外，从额外特召一只（？是否包括管家（二姐）
更衣
①：龙女仆的手融（从手卡 / 场上选融合素材）
②：在墓地发效果回手，并选场上一只本家怪弹回手
盛情
龙女仆的苏生，从手卡 / 墓地特召一只本家怪
还可以从卡组堆墓一张其他形态
（比如特召小绿堆墓大绿，特召大绿堆墓小绿）
迎接
①：己方全场攻守上升 100 × 场上的本家怪数量
②：己方场上存在 2 只以上本家怪时，
可以从墓地捡一张（迎接以外的）本家卡回手
③：此卡送墓的回合，本家怪不会成为效果对象
整理
①：选己方一只龙族怪兽弹回手卡，
同时从对方场上 / 墓地选一只怪弹回手（取对象）
②：墓地发效果除外，从手卡 / 墓地特召一只本家怪守备表示
（这只怪兽回合结束时，自动弹回手卡）
休息
选己方场上一只本家怪弹回手，两个效果二选一：
一、从卡组检索一张休息以外的本家卡
二、选对方场上一张魔陷弹回手

龙女仆下级只有粉绿红黑四姐妹吗？是不是忘了什么？
没有啊。小蓝？什么小蓝，那位不是表情包吗？咱龙女仆真不熟。

小蓝是在卡组和墓地之间运转资源的，没啥用（本家堆墓 / 检索 / 苏生能力已经够用了）。

至于小黑（暗属性）是真的没有龙形态（不存在大黑）。
龙女仆人气这么高，等以后（也许会有）的补强吧。参考隔壁闪刀姬

推荐外挂（小系统 / 单卡）

  • 暗视龙 特召暗属性 / 龙族怪兽时可以从手卡跳出
    众所周知，龙女仆起手是很怕康的，开局可以说康立停，但手里有暗视龙就可以挽回一点。
    通召本家下级，战吼吃康，直接变 撞针龙，然后特召暗视龙，白出一个 天球，还能过牌。
    而且可以当成融 大姐 的素材，拿在手里也不慌。哪怕拿去挡刀，2800 的防都还算可靠。
    进墓还有个保护里侧对象的效果，聊胜于无。这卡算是非常契合龙女仆主轴的怪兽了。
  • 伯吉斯异兽·高足杯虫 丢 1 张手卡（不是 cost）除外（取对象）1 张卡
    丢大龙很赚，反正战阶变身大龙，从手卡或者墓地拉都是一样的。
    墓地发效（连锁陷阱）返场是通常怪兽，可以变 星杯龙 做 天球。

可选额外

注意由于壶的存在，保险起见 二姐 和 天球 必下 3。其他都是喂壶的。

必选

  • 主动撞针龙 送墓配合 盛情（或 整理 的 ② 效果墓地发效）攀 link 值出天球
  • 天球之圣刻印 龙女仆先手其实很蠢，顺的话能站一个大姐，大部分时候大概就一个天球

可选

  • 交织绵羊 连接区有融合怪每回合可以复活墓地一只下级
    1. 攀 link 值：复活 小粉（盛情 / 大姐 / 二姐）拉另一只本家下级，出 刺刀 / 装弹
    2. 攀 link 值：复活 丝袜 跳 魔皇后 出 访问
    3. 拉一些奇奇怪怪的东西，比如 核成龙
    4. 攀打点：大逮捕 牛到同调 / 超量，3500（大姐） + 2000（大红） + 700（绵羊） 就能解 6000 的 到临
  • No.38 希望魁龙 银河巨神 R8 无效魔法牛过来当素材，拔素材嘲讽攻宣
    R8 只能用大红 + 大绿叠，但拿大龙来叠黑怪是非常亏的，注意自行取舍，下同。
  • 宵星之机神 丁吉尔苏 R8 战吼不取对象（吊打黄金卿）送墓（不是破坏），拔素材保护破坏

对局技巧 Tips

大红 人称小诚哥，作为手坑，有时候故意攻表本家下级（女仆）有奇效。
对方耿直地来撞可以血赚，大龙被解时（一般攻击力比较接近，< 2700+2000）也可以反杀。

注意大红作为全时点手坑，操作的时候千万不要手残用掉了。
比如发融合（更衣）会问时点发不发大红（就离谱，主阶我加攻干嘛，你问 nm 呢）。
这时候点是丢大红（加攻），然后选融合素材发现少了条大龙，人傻掉了。
我吃过好几次这个亏，只能说手太快了 手比脑子动得快。

大绿，虽然不是手坑，但这卡可以单防蛮多终端，比如高达。
（阿宙斯，就你 TM 叫天霆号是吧，翅膀给你拽掉，必须打你脸。）

手里没有上级但下级够多，多跳几次基本上都可以融出大姐，熟练就行。

盖好的整理可以躲泡影；整理 c1，休息 c2 两个效果都会结算，可以嫖一个 cost。

常用定式 Combo

  1. 开局通召 小绿 送 整理 入土
  2. 从墓地发动 整理 随便拉一只本家怪
  3. 场上的效果怪兽有两只，接下来做什么不用我多说了吧？
  4. DNA：收到收到。出来吧，青锁龙森蚺。森蚺 发效果检索 真红眼融合。出来吧，红爹！
  5. 游戏结束

MD 还没出红爹，那没事了。

玩笑结束，说回 combo。
龙女仆这个轴的特点就是慢热，属于典型的滚雪球体系（还不靠贴纸）。
赢面是靠本家下级怪兽（女仆）的战吼运转起来之后逐渐积累资源压死对面。
小绿堆墓任何本家，小黑检索本家魔陷，小红检索本家怪兽 + 堆墓本家怪兽，越打卡越多。
女仆进战阶可以弹回手卡并拉出手卡或墓地的本家大龙（变身本体）。
丢大绿 ban 怪兽发效果（不 ban 永续），大红当手坑加攻，大粉当手坑特召手里的下级（再配合本家魔陷弹回）。本家魔陷弹卡之后，更衣是从墓地回手，整理是弹对方一张（注意可以弹墓地[1]），休息可以「检索（休息以外的）任意本家」或者「弹对方一张后场魔陷」。
本家大龙结束战阶可以弹回手卡并拉出手里的女仆（变回人型）。
二姐在场的话己方每次龙族怪兽弹回手卡都可以炸对面一只怪，场上几个二姐每次就能炸几只。
没什么定式，熟练之后随机应变即可。

小黑 / 小绿单卡启动

  1. 通召 寝室龙女(小黑) 发效果检索 盛情
  2. 寝室龙女 变身（直接 LINK 召唤） 主动撞针龙（LINK-1）进墓 [2]
  3. 发动 盛情 苏生墓地的 寝室龙女
  4. 用 主动撞针龙 和 寝室龙女 LINK 出 天球之圣刻印
  5. 结束回合，到对方回合 天球之圣刻印 发效果解放自己弹一张卡（先选祭品再选弹）
  6. 解放的 天球之圣刻印 发效果从卡组拉出小绿 / 小红 堆墓大龙
    大龙堆下去女仆进战阶就可以直接变身拉起来
    • 小绿检索盛情（苏生）拉完还可以堆墓一张（拉的谁就堆谁，拉小绿堆大绿）
    • 小红检索 + 堆墓（手里有大龙可以检索别的，不然就检索大龙丢大龙）
  7. 记得龙够了之后第一时间（小黑）找 更衣 融大姐二姐，保证对场面的控制力
  8. 这个场面展开通过，手里攒住两张大龙，赢面就很大了
  9. 然后就是积累资源慢慢压死对面（伺机斩杀，推荐加入教导小系统，2 圣女 1 骑士即可）

小绿同理：

  1. 通召 客厅龙女(小绿) 发效果堆墓 整理
  2. 客厅龙女 LINK 出 主动撞针龙 进墓
  3. 墓地的 整理 发效果除外，苏生墓地的 客厅龙女
  4. 用 主动撞针龙 和 客厅龙女 LINK 出 天球之圣刻印
  5. 后面都一样

  1. 弹对面墓地有什么用？你是没挨过摆烂人 凤凰人 的打是吧（虽然目前 MD 还没出）。 

  2. 在 TCG/OCG 的环境里面，LINK-1 的 LINK 怪兽除了极个别如 独角兔、连接栗子球（注意区别于必需 电子界族 的 连环栗仔球）能够泛用，别的基本上都是为了送墓其他怪兽用的。
    至于 LINK-1 本身只需要特定的属性或种族能够纳入主轴的体系即可。这里的 撞针龙 就是典型，它是 LINK-1，还是 龙族，这就够了。它有什么效果并不重要，龙女仆也用不到什么「弹丸」字段。构筑需要它（甚至满三）只是为了靠 LINK 出它来「送墓 龙女仆的 本家下级怪兽」。
    另外，闪刀姬算是一朵奇葩，刀女人玩的就是 零衣 反复横跳拉各种 LINK-1 本家（以及通过 交闪 滤抽，这卡强度太离谱了。主要区域没怪 + 墓地 3 魔法的条件乍一看很苛刻，但你 TM 是闪刀姬啊，基本不用顾虑，别被康就行，然后「检索 1 抽 1」无副作用，堪比强欲之壶，甚至超过壶）。
